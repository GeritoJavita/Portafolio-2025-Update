APPETUD

Tecnologías utilizadas: Java 22.0, Java Swing, Metodología DAO y DTO, SQL Server, Tomcat.

Horas de trabajo: 65 horas Aprx.

La aplicación de escritorio "APPETUD" cuenta con CRUDs completamente funcionales y una base de datos optimizada para el manejo eficiente de la información. Además, incluye un manual de administración detallado para el cliente y una interfaz intuitiva que facilita su uso.

Git Hub Code

Vista Administrador - CRUD

CRUD - Panel Principal del Administrador

Vista principal donde el administrador accede a todas las funcionalidades: usuarios, reservas, propiedades, backup y más.

CRUD - Gestión de Clientes

Permite al administrador visualizar, editar o eliminar información de los clientes registrados.

CRUD - Gestión de Empleados

Visualiza y administra los datos de los empleados, incluyendo su rol y datos personales.

CRUD - Gestión de Reservas

Administra todas las reservas del sistema, incluyendo historial, estado y comentarios de los usuarios.

CRUD - Detalles de la Mascota

Permite revisar y actualizar la información específica de las mascotas registradas para cada cliente.

CRUD - Inventario General

Control y gestión del inventario de productos, servicios o propiedades disponibles en el sistema.

CRUD - Gestión de Servicios

Administra los servicios ofrecidos, permitiendo su creación, edición o desactivación según disponibilidad.

CRUD - Gestión de Equipos

Controla los equipos utilizados en los servicios o procesos, con opciones para actualizar sus estados.

CRUD - Servicios Médicos

Sección para gestionar servicios médicos asociados a las mascotas, como tratamientos o consultas.

CRUD - Gestión de Facturas

Permite al administrador ver, generar y modificar facturas emitidas por los servicios realizados.

CRUD - Gestión de Pagos

Visualiza los pagos realizados por los usuarios y administra su estado o reembolsos necesarios.

CRUD - Acceso y Autenticación

Módulo de inicio de sesión que permite el acceso seguro al sistema por parte de los administradores.
